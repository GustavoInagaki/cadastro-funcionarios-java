# Cadastro de Funcionarios em Java

Projeto simples desenvolvido para prática de lógica de programação utilizando Java.

## Funcionalidades
- Cadastro de pessoas
- Listagem de pessoas
- Armazenamento em memória (ArrayList)

## Conceitos aplicados
- Estrutura de repetição (do-while)
- Estrutura condicional (switch)
- Entrada de dados com Scanner
- Listas em memória
